# アイリスオーヤマ プレミアリーグU-11 公式アプリ
## スポンサー提案資料

---

## 1. アプリ概要

**アイリスオーヤマ プレミアリーグU-11 公式PWAアプリ**は、全国のU-11サッカーリーグに参加する選手・保護者・コーチ向けの総合プラットフォームです。

### 基本情報
- **アプリ形式**: PWA（Progressive Web App）
- **対応端末**: スマートフォン（iOS/Android）、タブレット、PC
- **本番URL**: https://u11-premier-pwa.vercel.app
- **技術スタック**: Next.js 15, React 18, Supabase, TailwindCSS

### ターゲットユーザー
| ユーザー種別 | 想定人数 | 主な利用シーン |
|------------|---------|--------------|
| 選手（小学5年生） | 約5,000人/年 | 試合確認、選手証提示、成長記録 |
| 保護者 | 約10,000人/年 | スケジュール確認、出欠回答、試合観戦 |
| コーチ・指導者 | 約1,000人/年 | チーム管理、試合結果入力、選手育成 |
| リーグ運営者 | 約50人/年 | 試合管理、順位表管理、お知らせ配信 |

## 2. 実装済み機能一覧

### 2.1 リーグ情報（全ユーザー向け）

#### ダッシュボード（ホーム画面）
- リーグロゴ・チームロゴ表示
- 今週の予定（練習・試合）
- 今週の試合一覧（LIVE表示対応）
- リーグ順位表（TOP5）
- お知らせ一覧（YouTubeサムネイル対応）
- **スポンサーバナー表示エリア**

#### 試合情報
- 試合一覧（日程・結果）
- 試合詳細（スコア、タイムライン、会場情報）
- LIVE試合のリアルタイム更新

#### リーグ順位表
- 全チーム順位表示
- 勝点・得失点差・直接対決成績
- チームロゴ表示

#### チーム一覧
- 参加チーム情報
- チームロゴ・所属地域

### 2.2 デジタル選手証

#### 機能概要
- **QRコード付きデジタル選手証**
- 試合会場での本人確認に使用可能
- 紙の選手証を代替するペーパーレス化

#### 表示情報
- 選手氏名
- 所属チーム・チームロゴ
- 背番号・ポジション

- 登録シーズン
- 所属リーグ（神奈川2部A等）
- 登録ID
- QRコード（選手情報へのリンク）

#### スポンサー露出ポイント
- 選手証ヘッダーにリーグロゴ
- 選手証フッターにスポンサー名表示可能

#### ホーム画面追加
- スマートフォンのホーム画面にアプリアイコン追加可能

## 5. 技術的特徴

### 5.1 PWAの利点

- **インストール不要**: URLアクセスで即利用開始
- **自動更新**: アプリストア審査不要で即時更新
- **クロスプラットフォーム**: iOS/Android/PC全対応
- **低コスト運用**: App Store/Google Play手数料なし

### 5.2 セキュリティ

- HTTPS通信
- Supabase Row Level Security
- 認証・認可の厳格な管理
- 個人情報保護対応

### 5.3 スケーラビリティ

- Vercel Edge Network による高速配信
- Supabase PostgreSQL による堅牢なデータ管理
- 全国展開にも対応可能なアーキテクチャ
