## アイコン作成方法

1. https://realfavicongenerator.net/ にアクセス
2. マスター画像（512x512px以上）をアップロード
3. 全サイズを自動生成
4. ダウンロードして `public/icons/` に配置

または、デザイナーに依頼して適切なブランドアイコンを作成してもらってください。
